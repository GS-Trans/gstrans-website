## 🔧 Customization Options

### To Change Colors
Edit [tailwind.config.js](tailwind.config.js):
```javascript
colors: {
  primary: {
    50: '#e6f4ff',   // Lighter blue
    600: '#1890ff',  // Main blue (change this)
    800: '#0066cc',  // Darker blue
  }
}
```

### To Update Content
Edit [data/CONTENT.md](data/CONTENT.md) - all text in one place!

### To Add Pages
Create new files in `src/app/`:
- `src/app/about/page.tsx` → /about
- `src/app/services/page.tsx` → /services
- `src/app/contact/page.tsx` → /contact
